# Portfolio

New and improved personal website, written in React.

## Setup

First, install `nvm` to manage node versions: https://github.com/nvm-sh/nvm.

```bash
nvm install 16
nvm use 16
npm install
```

## Running
```bash
nvm use 16 # Doesn't work with node > 16.
npm start
```

## Deploying
```bash
nvm use 16 # Doesn't work with node > 16.
npm run deploy
```
